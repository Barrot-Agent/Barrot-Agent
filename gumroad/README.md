# Gumroad Integration Directory

This directory contains files related to Gumroad integration.

## Directory Structure

- **`products/`** - Product configurations and definitions
- **`webhooks/`** - Webhook handlers for Gumroad events
- **`api/`** - API integration code for Gumroad

## Getting Started

See the main [GUMROAD.md](../GUMROAD.md) file in the root directory for detailed documentation on:
- Setting up Gumroad integration
- Configuration requirements
- Security best practices
- API usage examples

## Quick Reference

- Products: Place product JSON/YAML files in `products/`
- Webhooks: Implement webhook handlers in `webhooks/`
- API: Add API client code in `api/`

For questions or support, refer to the main documentation.
